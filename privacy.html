Какие данные мы собираем

При регистрации мы сохраняем:

  • Имя пользователя
  • Адрес электронной почты
  • Идентификатор Telegram (если вход через Telegram)

Мы также сохраняем прогресс обучения (пройденные блоки, дата окончания доступа).

Как мы используем данные

  • Для управления доступом к материалам приложения
  • Для отправки уведомлений об обучении (по вашему согласию)
  • Для технической поддержки

Передача данных третьим лицам

Мы не продаём и не передаём ваши данные третьим лицам. Данные хранятся в защищённой базе данных Supabase (supabase.com).

Микрофон

Приложение использует микрофон для распознавания речи. Аудио обрабатывается через Deepgram API и не сохраняется.

Удаление данных

Вы можете запросить удаление вашего аккаунта и всех связанных данных. Подробнее: страница удаления аккаунта.
